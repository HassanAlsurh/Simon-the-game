About Simon

Simon is a classic memory game. The game has four colored buttons. It will play a pattern of lights and sounds. Your goal is to remember the pattern and repeat it back. Every time you get it right, the pattern gets one step longer!

Directions

  1. Load the deployed game
  2. Press START
  3. Press the buttons in the same sequence that they light up
  4. Press the ON button to pause
  5. Press the OFF button to resume
  6. If you lose or win, press RESTART to restart the game
