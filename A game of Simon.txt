A game of SimonPseudocode for the game:- Create a circle with 4 buttons each with a distinguished color- Set the sequence array- check current game mode if training is on, ignore restart condition upon failing- play the first level of the sequence- wait for user to press the buttons- compare pressed button to the played sequence	- if correct proceed to next level of the sequence and increase the score var	- if wrong, check game mode, if training repeat current level sequence, if 'off' compare current score to 'highest score' , save the biggest to the highest score var, and then restart the sequences from level 0.- upon reaching score of 32, the user is a winner, play 'win' sound effects and display the user has won!Variables:- multidimensional array for the sequences (each index is a level and each level will have its own sequence)- a variable to listen to user clicks and compare it to the current sequence index- a count variable that get incremented each win- another count variable that get incremented with each wrong input of the user (adding a lose condition)- current level variablefunctions:- a math function to add a variation in the timing (asynchronous timing)- a custom function to use to compare user inputs to the sequenceUser Stories:As a user I ... so that ...- aau I want to see instructions so that I can know how to
